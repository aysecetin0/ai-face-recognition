# ai-face-recognition
İleri teknolojiyle yüzleri anında tanıyın! Python ve `face_recognition` tabanlı bu proje; gelişmiş yüz algılama, gerçek zamanlı kamera takibi ve kullanıcı dostu modern bir arayüz sunar. Kişilerin yüz haritalarını çıkararak `encodings.pkl` dosyasında saklar ve canlı yayında veya fotoğraflarda %99'a varan doğrulukla eşleştirme yapar.
